设计模式-代理模式

2021-02-23
设计模式
约 326 字 预计阅读 1 分钟

文章目录

定义 为其他对象提供一个代理通道，来完成对这个对象的访问。
类型 结构型模式
特点 可以对调用者隐藏实际对象。
举例 张三替李四追求王五。

本文以C#为例，先来看类图🔻

代理模式还是相对比较简单和容易理解的，所以我们直接看具体的代码实现🔻

    /// 定义一个代理和真实类共用的接口

    /// <summary>
    /// 代理类
    /// </summary>
    public class ProxyLove : LoveBase
    {
        private RealLove _realLove;
        public override void SendLove()
        {
            //初始化真实的表白对象，并调用真实的表白方法
            _realLove ??= new RealLove();
            _realLove.SendLove();
        }
    }
