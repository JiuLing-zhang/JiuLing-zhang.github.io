SQL查询用逗号分隔的数据

Veröffentlicht am 2021-02-04 in SQL Server

需求

不同的商品使用逗号分隔保存，现在要求输入指定的商品名称，能够查询到包含该商品的列。

  • 输入 K701 返回第1、2行
  • 输入 K702 返回第2、4行
编号 商品
1 K701
2 K701,K702
3 K701B,K701C
4 2K701,K702

解决思路

  • 方法1：将商品列按照逗号分隔后列转行，然后在转换后的集合里查找符合条件的商品，最后用行ID去原始表反查
  • 方法2: 将每一个商品名称格式化成 ,*, 形式，然后使用like查询 like '%,*,%'
